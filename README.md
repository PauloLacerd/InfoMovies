# InfoMovies

#### Foi utilizado Angular e Bootstrap para a construção da aplicação. Ela consome a API do The Movie DB(https://www.themoviedb.org/).

#### É possivel pesquisar um filme especifico para observar os seus dados como também é possivel observar os dados de um filme do catálogo. A aplicação possui opções de acessibilidade como aumentar e diminuir a fonte além da opçao de auto contraste.

#### A aplicação utiliza rotas para não precisar de refresh durante a navegação e usa uma classe isolada de serviços para o consumo da API.
